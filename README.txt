## あなただけの学習ロードマップ
### Phase 1: フロントエンドの世界へ飛び込もう！ (HTML, CSS, JavaScript)

- **期間**: 1週目〜3週目
- **目標**: Webページの基礎を築き、JavaScriptで動的なページを作成できるようになる
- **内容**:
    * **第1週**: Webページの構造を理解し、HTMLを使ってシンプルなToDoリストの外観を作ってみましょう。
        * 学習内容：HTMLの基本、タグ、属性、構造など
        * 課題：自己紹介ページの作成
    * **第2週**: CSSを使って、ToDoリストをおしゃれにデザインしましょう！
        * 学習内容：CSSの基本構文、セレクタ、プロパティ、CSSフレームワークなど
        * 課題：ToDoリストのデザイン改善
    * **第3週**: JavaScriptの基礎を学び、ToDoリストに「項目の追加機能」を実装しましょう！
        * 学習内容：JavaScriptの基本構文、変数、データ型、条件分岐、関数、DOM操作など
        * 課題：JavaScriptでインタラクティブな要素を追加 (ボタン、モーダルなど)
- **学習教材**: 各週ごとに動画教材とテキスト教材、サンプルコードを提供します。
- **課題提出**: 毎週日曜日の23:59までに提出してください。
- **質問**: オンラインフォーラムで随時受け付けます。

### Phase 2: React & TypeScriptで開発効率UP! (React, TypeScript)

- **期間**: 4週目〜7週目
- **目標**: ReactとTypeScriptを習得し、効率的な開発手法を身につける
- **内容**:
    * **第4週**:  JavaScriptをより安全に、効率的に書くためのTypeScriptを学びましょう！
        * 学習内容：TypeScriptの基本構文、型注釈、インターフェースなど
        * 課題：ToDoリストの一部機能をTypeScriptで書き換え
    * **第5週**:  Web開発で必須のフロントエンドライブラリ、Reactの基本をマスターしましょう！
        * 学習内容：Reactの基本概念、JSX、コンポーネント、props、stateなど
        * 課題：Reactでシンプルなコンポーネントを作成
    * **第6週**:  ReactとTypeScriptを組み合わせて、ToDoリストをさらに機能アップしましょう！
        * 学習内容：コンポーネントの分割、イベント処理、useState、useEffectなど
        * 課題：ReactとTypeScriptを用いてToDoリストを再構築
    * **第7週**:  React Routerを使って、ToDoリストに複数のページを追加しましょう！
        * 学習内容：ルーティング、Link、useParams、useNavigateなど
        * 課題：ToDoリストにページを追加 (完了済みページなど)
- **学習教材**: 各週ごとに動画教材とテキスト教材、サンプルコードを提供します。
- **課題提出**: 毎週日曜日の23:59までに提出してください。
- **質問**: オンラインフォーラムで随時受け付けます。

### Phase 3: アプリの裏側を作ろう！ (バックエンド＆インフラ)

- **期間**: 8週目〜12週目
- **目標**: バックエンドの基礎を学び、APIサーバーとデータベースを連携させて、ToDoリストをより本格的なアプリに進化させよう！
- **内容**: 
    * **第8週**: Node.jsとExpressを使って、自分のAPIサーバーを構築してみましょう！
        * 学習内容：バックエンドの役割、APIサーバーの基礎、Node.js、Express、ルーティングなど
        * 課題：Node.jsとExpressでシンプルなAPIサーバーを構築
    * **第9週**: データベースを使って、ToDoリストのデータを永続化しましょう！
        * 学習内容：データベースの基本概念、RDBMS、SQL、データベース接続、データ操作など
        * 課題：APIサーバーにデータベース(SQLiteなど)を接続し、ToDoデータのCRUD処理を実装
    * **第10週**: API設計の基礎を学び、より使いやすいAPIを設計・開発しましょう。
        * 学習内容：RESTful API、API設計原則、HTTPステータスコードなど
        * 課題：ToDoリストのAPI仕様を定義し、APIサーバーに実装
    * **第11週**: フロントエンドとバックエンドを繋げて、データのやり取りを実装しましょう！
        * 学習内容：API通信 (fetch, axiosなど)、非同期処理、データの受け渡し (JSON)、エラーハンドリング
        * 課題：React製のToDoリストを、開発したAPIサーバーと連携させる
    * **第12週**: ついに、開発したToDoリストアプリをインターネット上に公開しましょう！
        * 学習内容：デプロイ、PaaS、Heroku、環境構築、デプロイ手順、ドメイン設定
        * 課題：開発したToDoリストアプリケーションをHerokuなどのPaaSにデプロイし、インターネット上に公開
- **学習教材**: 各週ごとに動画教材とテキスト教材、サンプルコードを提供します。
- **課題提出**: 毎週日曜日の23:59までに提出してください。
- **質問**: オンラインフォーラムで随時受け付けます。
